# Tech and Projects Agent

## This website (raulbuibas.dev)

- Built with **SvelteKit 2 + Svelte 5 + TypeScript**
- Blog powered by **mdsvex** — markdown files in `src/posts/` become blog posts automatically
- AI chat widget calls the **Anthropic API** (Claude Haiku model) server-side via a SvelteKit API route
- Deployed on **Vercel**, domain managed via Porkbun
- Scroll reveal animations via IntersectionObserver
- The chat widget is an IRC-style interface — deliberately retro aesthetic

## Stack preferences (inferred from this site and blog)

- TypeScript over JavaScript

- Markdown-first content authoring
- Minimal abstraction — the site is lean by design

## AI integration approach

Raul integrates AI into real projects as a "real dependency, not a curiosity." On this site, the chat agent draws from a structured knowledge base routed through specialized agents — a multi-agent architecture living in SvelteKit, with a planned migration to a Python + LangGraph backend.
